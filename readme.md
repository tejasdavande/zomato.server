# Zomato Server

## About the API

This was the zomato online food order app.
It was created on the Rest API platform.

## How To Run Server Locally

**1. npm install**

**2. create new **.env** file.**

**3. Add Below mentioned details in **.env** file to run.**

> `DB_username`

> `PORT`

> `JWT_SECRET_KEY`

> `DB_password`

> `DB_name`

**4. npm run dev**
